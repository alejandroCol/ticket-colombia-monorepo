# 📐 Especificaciones para el Logo de Ticket Colombia

## 🎯 Logo Principal (Navbar/Header)

### Archivo Actual
- **Nombre**: `logo.png`
- **Ubicación**: 
  - `/bitcomedia-main-app/src/assets/logo.png`
  - `/bitcomedia-web-admin/src/assets/logo.png`

### Especificaciones Técnicas
- **Dimensiones**: `985 x 256 pixels` (ancho x alto)
- **Ratio**: `3.85:1` (horizontal/alargado)
- **Formato**: PNG con transparencia (RGBA)
- **Peso**: ~25 KB
- **Color depth**: 8-bit

### Dimensiones de Visualización
- **Desktop**: Se muestra a `40px` de alto (el ancho se ajusta proporcionalmente)
- **Mobile**: Se muestra a `35px` de alto

## 🎨 Recomendaciones de Diseño

### Para el Nuevo Logo "Ticket Colombia"

**Dimensiones Recomendadas**:
- **Ideal**: `1000 x 260 pixels` (para mantener ratio similar)
- **Mínimo**: `800 x 208 pixels` (calidad aceptable)
- **Máximo**: `2000 x 520 pixels` (retina/alta resolución)

**Formato**:
- PNG con fondo transparente (RGBA)
- Comprimir a menos de 50 KB para web

**Colores del Tema Dark Blue**:
- **Background oscuro**: `#0d1b2a` (donde se verá el logo)
- **Accent cyan**: `#00d4ff` (para elementos destacados)
- **Texto claro**: `#e0e1dd` (para texto en el logo)
- **Blue-gray**: `#778da9` (para elementos secundarios)

**Estilo Sugerido**:
- Logo horizontal (más ancho que alto)
- Texto claro sobre fondo oscuro
- Usar el cyan (`#00d4ff`) para acentos
- Diseño limpio y moderno
- Buena legibilidad en tamaños pequeños (40px alto)

## 🔄 Cómo Reemplazar el Logo

1. **Diseña** el logo con las especificaciones de arriba
2. **Exporta** como PNG con transparencia
3. **Optimiza** el archivo (usar TinyPNG o similar)
4. **Renombra** a `logo.png`
5. **Reemplaza** en ambas carpetas:
   - `/bitcomedia-main-app/src/assets/logo.png`
   - `/bitcomedia-web-admin/src/assets/logo.png`
6. **Recarga** la aplicación (Cmd + Shift + R)
