# Ticket Colombia - Panel Administrativo

Este proyecto es el panel administrativo de Ticket Colombia. Construido con React, TypeScript y Vite, esta aplicación proporciona una interfaz de usuario moderna y eficiente para gestionar el contenido y servicios de Ticket Colombia.

## Technologies

- [React 19](https://react.dev/) - Frontend library
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Vite](https://vitejs.dev/) - Build tool and development server
- [Firebase](https://firebase.google.com/) - Backend services
- [React Router](https://reactrouter.com/) - Routing
- [SASS](https://sass-lang.com/) - Styling

## Getting Started

### Prerequisites

- Node.js (latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd bitcomedia-web-admin
```

2. Install dependencies
```bash
npm install
# or
yarn
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
```

## Available Scripts

- `npm run dev` - Starts the development server
- `npm run build` - Builds the app for production
- `npm run lint` - Runs the linter
- `npm run preview` - Previews the production build locally

## Development Guidelines

- Follow the established component patterns in `src/components`
- Use SASS for styling
- Implement responsive design for all UI components

## Deployment

The application is built using Vite's build system:

```bash
npm run build
```

This will generate optimized files in the `dist` directory that can be deployed to your hosting service of choice.
